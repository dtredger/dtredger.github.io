Showroom - github

Showspace is a pretty standard Rails 4.1 crud app that allows users to view items in a grid interface and save them to a "closet" for later. Check the github readme for more info.

Uses Devise for authentication, carrierwave for images (uploaded to s3), rspec + FactoryGirl for tests. Tests are mostly around Model and controller tests (only a few request specs, since we aren't really settled on the ui).

There is an admin interface (that uses active-admin), which allows for managing items (batch-deleting, checking whether items are duplicates, etc.

The master branch is running on heroku.

WeatherTF - github

WeatherTF is a simple rails app that sends you an SMS message with the forecast for your latitude/longitude every morning. Lat/Lon is determined by browser geolocation (if available).

Messages are actually just emails, which are sent to your carrier's SMS gateway, which will deliver a plain-text email as a message. Messages are sent at 8am (EST) every morning via resque-scheduler (normal emails use resque).

Forecasts are via the Forecast.io. API.

Sample running on heroku. Go check out the background-images (from Ansel Adams' "National Parks" series), but don't sign up, because there is no way to stop getting messages. Work in progress.
